About This Project

An ML-powered tool to estimate household poverty levels from survey data.

üìã Overview

This tool uses an XGBoost regression model trained on household survey data to predict per-capita daily utility expenditure (in PPP dollars). The prediction is then mapped to internationally recognised poverty thresholds set by the World Bank.

üî¨ Methodology

  • Data Cleaning: KNN imputation for numeric columns; mode imputation for categorical columns.
  • Feature Engineering: Categorical features encoded to numeric codes; outlier removal via IQR method.
  • Scaling: StandardScaler applied to all features.
  • Model: XGBoost Regressor with 1 000 estimators, RMSE evaluation metric, and early stopping (50 rounds).
